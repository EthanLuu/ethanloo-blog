好物分享 - 终端 | 封面 | 快捷方式

发表于2021-06-07|黑科技
|字数总计:808|阅读时长:2分钟

好久没来分享黑科技了，趁今天晚上课设写累了，分享一些自己最近搜刮到的好东西。

先声明，文章中推荐的软件只是我用了一段时间之后觉得很好用，不保证适合你。

Windows Terminal：好看的终端不只是皮囊

还在用古老的黑乎乎命令行窗口？改用 Windows PowerShell 了？

看看新版的 Windows 命令行界面吧。

毛玻璃，自定义字体，透明度，自定义配色…光这些表面功夫就已经够吸引人了。让自己写代码的时候保持心情愉悦何尝不是提升效率呢？

你甚至还可以直接在里面配置不同的命令行窗口，包括一些 Ubuntu 子系统等。

Win10 应用商店自行安装完之后通过搜索引擎找一下美化方案吧！

Figma：设计封面？抄作业就完事儿了！

在前端技术相当成熟的今天，已经有不少的 Web 应用相当好用，在舍弃繁琐的安装步骤之后，显得更加强大。

在接触到 Figma 之前，我一直在 fotor 上制作我的封面，正如下图所示，这是我之前做过的一些图。

虽说这个网站已经足够方便制作封面了，只需要改改模板上的文字就是一张完整的封面图。

奈何人总是贪婪（吝啬？）的，这个网站的大多数模板都是收费的（VIP），免费的模板少的可怜。

网站甚至不提供 「只显示免费」的选项

于是，我便找到了一个新的宠儿，让我们有请 FIGMA ！这是我的仓库，其中有一些最近的封面。

严格意义上来讲，这个应用面向的群体并不是简单的封面图需求者，而是真正意义设计师。

不过，我作为一个没啥美术造诣的人，虽不会创作，但是抄作业总还是会的。

进入社区，可以看到很多设计师分享的设计作品，根据他们的作品和自己的需求，我凭借自己还 OK 的审美，勉强能做个简洁舒服的封面。

最近因为一些众所周知的原因，很多网站被墙了，包括 Figma 😓

uTools：强迫症福音 程序员之友

先分享一下我的桌面：

在桌面上我只放了我使用频率最高的东西，包括底下的菜单栏。

你很好奇我是怎么打开其他软件的对不对？（不对）

有的时候，我用 Win10 的菜单栏，其实挺好看的，就是找东西有点费眼睛。

不过最近用过 uTools 后，我已经准备慢慢淘汰这种方法了。

给你看下这个软件的使用截图，你就知道我为什么把他叫做程序员之友了。

看起来是不是有点像什么东西？对！就是命令行。

这就是我使用频率最高的功能，在任意界面通过 『Alt + 空格』呼出命令行，输入关键字或者首字母，快速打开软件。

当然，它也提供了不少的扩展功能。
