M1 MacBook Pro 20天初体验

发表于2021-09-23|日记
|字数总计:776|阅读时长:2分钟

购买前

从七月份开始，由于原来那台小米笔记本偶尔会出现蓝屏的情况，就已经开始犹豫要不要换笔记本了。但是，本质上我开发使用的是公司配的台式机，所以并没有什么紧迫感，只略微增加了一点平常浏览电子产品测评的视频的时间。

最终，随着八月份蓝屏的频率越来越高，还是咬咬牙买了2020款的M1 MBP。

花了两天时间纠结购买途径和配置，逛遍了知乎、B 站，根据自己的需求：

  • 中轻度的前端开发
  • 需要使用虚拟机
  • 基本用不到视频剪辑

确定了要买 16 GB + 256 GB。

购买时

第一次自己购买贵重的电子产品，本来以为可以第一次体验一下花呗分期的快乐（毕竟当时自己才拿了一个半月的工资，还付了房租）。结果发现，我的额度只有2k多，完全没办法用来分期买电脑 👐，无奈之下还是靠爸妈打钱买下了电脑。

十月份补充，已经把我爸给我打的 8k 全部还给他了。

虽然下单了，但是还是要感叹一句太贵了吧！！！

购买后

两个字，满意。

高素质的屏幕，接近 12H 的电池续航，简单纯粹的系统体验，B 格拉满的 Macgic Bar，丝滑流畅的触摸板……

这是目前装的一些常用软件，以后有机会挑几个好用的详细介绍一下。

抛开优点，这里展开讲讲使用体验上给我感觉有些糟糕的地方。

Finder

绝了，我第一次发现 Windows 资源管理器是这么好用。在 Finder 里找个类似我的电脑的东西都找了老半天。

快捷键

这个或许不是 MacOS 本身的问题，而是 Windows 用户在适应 MacOS 过程中必然遇到的一个难于跨越的鸿沟。

没有 Ctrl 就算了，我可以用 Command，但是谁能告诉我为啥有的时候是用 Command 替代，有的时候又变成了 Option 来替代 Ctrl？？？

App Store

第一次登录 App Store 失败，我以为是网络问题没有深究，毕竟软件可以通过官网下载 dmg 安装包。

第二次公司网登录失败，仍然继续忽略。

接着又是 N 次登录失败，直到有一次我需要下载 Microsoft Remote Desktop 必须要访问 App Store 时，我才去尝试解决这个问题。

参考网上的解决方案，更换热点，连接代理，更换节点，终端输入命令……都没能成功解决问题，最终我重新登录了一次 Apple ID 才成功进入了 App Store。

20 天初次体验后的建议

  1. M1 Macbook Pro 基本没有不兼容问题。
  2. 适合不喜欢带充电器的人群。
  3. 笔记本自带 2 个雷电接口，如果对接口有更高需求，建议入手雷电 3 拓展坞。
  4. 早买早享受，晚买有折扣。
